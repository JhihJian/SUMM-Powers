# External Skills

此目录存放从 [antigravity-awesome-skills](https://github.com/sickn33/antigravity-awesome-skills) 加载的外部技能。

## 使用方法

使用 `/summ:find-skill <关键词>` 搜索并加载技能。

## 调用已加载的技能

```
summ:external:<skill-id>
```

## 目录结构

```
external-skills/
├── .gitkeep
├── README.md
└── <skill-id>/
    └── SKILL.md
```

## 注意事项

- 加载的技能保留供后续使用
- 注意查看技能的 risk 级别
- critical 级别技能请谨慎使用
